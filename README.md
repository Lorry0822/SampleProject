# SampleProject
--------------
## 一个添加了基本通用库和部分针对性工具的模板项目
## 使用 WanAndroid 开放 API

### 使用架构
* MVVM + Jetpack + DataBinding
* 网络请求 Retrofit + 协程
* 图片加载 Glide
* 刷新框架 SmartRefreshLayout
* 数据存储 MMKV
